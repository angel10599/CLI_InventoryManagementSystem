## 📌 Description

This is a Command-Line Interface (CLI) based Inventory Management System developed using C#.
The system allows users to manage products, categories, suppliers, and transactions efficiently through a simple console application.

---

## 🎯 Features

* Add, view, update, and delete **Products**
* Manage **Categories**
* Manage **Suppliers**
* Record and track **Transactions**
* Login system with role-based access (Admin & Staff)
* Input validation
* Organized code using OOP principles

---

## 🛠️ Technologies Used

* C#
* .NET Framework / .NET Core
* Console Application

---

## 📂 Project Structure

* `Program.cs` – Main system flow and menu
* `Product.cs` – Product model
* `Category.cs` – Category model
* `Supplier.cs` – Supplier model
* `TransactionRecord.cs` – Transaction handling
* `User.cs` – User model

---

## ▶️ How to Run the System

1. Open the project in **Visual Studio**
2. Build the solution
3. Run the program (Press **F5**)
4. Login using the credentials above
5. Follow the menu options
